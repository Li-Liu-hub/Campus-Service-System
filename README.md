# 🏫 校园服务系统 (Campus Service System)

## 🎯 项目定位

本项目是一个基于 **Spring Boot + Vue3** 构建的校园任务撮合与社区互动平台。
抛弃了传统的"纯 CRUD"堆砌，本项目致力于解决真实业务场景下的核心痛点，围绕**订单撮合、帖子交流、实时私信、评价信誉、搜索排序与后台治理**构建了完整的业务闭环。结合 Redis、RabbitMQ、WebSocket、MySQL 索引优化等技术，实现了高可用、高性能的工程化落地。

## 💡 核心技术亮点 (Core Features)

- **🔄 订单生命周期与状态机闭环**
  - 摒弃简单的下单逻辑，实现完整的订单状态流转（发布 -> 接单 -> 完单/取消），并支持复杂的取消溯源与时间线记录。
- **⭐ 双向评价与信誉体系**
  - 设计并实现了订单完单后的双方互评功能，底层联动用户信誉分计算，建立真实的社区信任体系。
- **🔍 全文搜索与热度排序引擎**
  - 针对帖子与订单需求，基于 MySQL 全文索引（Full-Text Index）重构搜索逻辑。
  - 引入自定义热度排序规则（结合浏览量、回复数与新鲜度加权），并使用 Redis 缓存热门搜索词，大幅降低数据库压力。
- **🛡️ 举报审核与内容治理工作流**
  - 从源头解决社区内容生态问题，搭建"用户举报 -> 自动阈值拦截 -> 后台人工审核（通过/驳回）"的完整治理闭环。
- **💬 高可靠 WebSocket 实时通信**
  - 升级私信模块，支持 WebSocket 实时消息与图片传输（复用 OSS）。
  - 重点解决网络抖动场景下的**消息发送确认（ACK）**、**消息幂等性**以及**离线未读数同步**问题。

## 🛠️ 技术栈

* **后端:** Spring Boot, MyBatis-Plus, Spring Security, JWT
* **中间件:** Redis (热点缓存), RabbitMQ (异步解耦), WebSocket (实时通信)
* **数据库:** MySQL (全文索引与查询调优)
* **前端:** Vue3, TypeScript, Element Plus
* **云服务:** 阿里云 OSS

## 快速启动

### 环境要求

- JDK 17+
- MySQL 8.x
- Redis 6.x
- RabbitMQ 3.x
- Node.js 18+
- Maven 3.9+

### 后端启动

1. 导入 `jsyl_ms.sql` 初始化数据库
2. 修改后端配置文件中的 MySQL、Redis、RabbitMQ 连接信息
3. 进入 `Jsyl-springboot-background/` 目录
4. 执行 `mvn clean install`
5. 启动后端主服务

### 前端启动

1. 进入 `Jsyl-springboot-vue3-front/` 目录
2. 执行 `npm install` 安装依赖
3. 执行 `npm run dev` 启动前端
