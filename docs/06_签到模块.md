# 签到模块

## 模块简介

签到模块提供每日签到功能，鼓励用户活跃度，可设置连续签到奖励机制。

## 功能清单

### 1. 每日签到
- 每日签到一次
- 连续签到计数
- 签到奖励发放

### 2. 签到记录
- 查看历史签到记录
- 查看连续签到天数

### 3. 签到统计
- 本月签到天数
- 最长连续签到记录

### 签到 Service
```java
/**
 * 签到服务实现
 */
@Service
public class SignInServiceImpl implements SignInService {

    @Autowired
    private SignInMapper signInMapper;

    /**
     * 签到
     * @param userId 用户ID
     * @return 签到结果VO
     */
    @Override
    public SignInVO signIn(Integer userId) {
        LocalDate today = LocalDate.now();

        // 1. 检查今天是否已签到
        SignIn todaySignIn = signInMapper.getByUserIdAndDate(userId, today);
        if (todaySignIn != null) {
            throw new BusinessException("今日已签到");
        }

        // 2. 计算连续签到天数
        LocalDate yesterday = today.minusDays(1);
        SignIn yesterdaySignIn = signInMapper.getByUserIdAndDate(userId, yesterday);

        int continuousDays = 1;  // 初始为1天
        if (yesterdaySignIn != null) {
            // 昨天已签到，累加连续天数
            continuousDays = yesterdaySignIn.getContinuousDays() + 1;
        }

        // 3. 计算签到奖励（连续签到天数越多，奖励越多）
        int points = calculatePoints(continuousDays);

        // 4. 保存签到记录
        SignIn signIn = SignIn.builder()
                .userId(userId)
                .signDate(today)
                .points(points)
                .continuousDays(continuousDays)
                .createTime(LocalDateTime.now())
                .build();
        signInMapper.insert(signIn);

        // 5. 返回签到结果
        return SignInVO.builder()
                .signDate(today)
                .points(points)
                .continuousDays(continuousDays)
                .totalPoints(getTotalPoints(userId))
                .build();
    }

    /**
     * 计算签到奖励积分
     * 连续签到奖励机制：
     * - 第1天: 10积分
     * - 第2天: 20积分
     * - 第3天: 30积分
     * - ...
     * - 连续7天后重置
     */
    private int calculatePoints(int continuousDays) {
        int day = (continuousDays - 1) % 7 + 1;  // 1-7循环
        return day * 10;
    }

    /**
     * 获取用户当前连续签到天数
     */
    @Override
    public Integer getContinuousDays(Integer userId) {
        LocalDate today = LocalDate.now();
        SignIn todaySignIn = signInMapper.getByUserIdAndDate(userId, today);
        if (todaySignIn != null) {
            return todaySignIn.getContinuousDays();
        }

        // 今天未签到，检查昨天
        LocalDate yesterday = today.minusDays(1);
        SignIn yesterdaySignIn = signInMapper.getByUserIdAndDate(userId, yesterday);
        if (yesterdaySignIn != null) {
            return yesterdaySignIn.getContinuousDays();
        }

        return 0;
    }

    /**
     * 获取签到记录
     */
    @Override
    public List<SignIn> getSignInHistory(Integer userId, Integer limit) {
        return signInMapper.getHistoryByUserId(userId, limit);
    }
}
```

## 数据库表

### sign_in（签到表）
```sql
CREATE TABLE `sign_in` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '签到ID',
  `user_id` int NOT NULL COMMENT '用户ID',
  `sign_date` date NOT NULL COMMENT '签到日期',
  `points` int DEFAULT 0 COMMENT '签到获得积分',
  `continuous_days` int DEFAULT 1 COMMENT '连续签到天数',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_user_date` (`user_id`, `sign_date`),
  INDEX `idx_user_id`(`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='签到记录表';
```
