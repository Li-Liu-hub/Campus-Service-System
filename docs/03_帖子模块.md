# 帖子模块

## 模块简介

帖子模块是校园服务平台的核心功能之一，允许用户发布校园信息、交易闲置物品、发布活动等，并支持评论互动。

## 功能清单

### 1. 帖子发布
- 选择帖子类型（校园交易、闲置物品、失物招领、问答求助、活动约伴）
- 填写标题和内容
- 上传图片（最多9张）
- 敏感词过滤

### 2. 帖子浏览
- 分页查询帖子列表
- 按类型筛选
- 按热度排序
- 推荐算法排序

### 3. 帖子详情
- 查看帖子完整内容
- 查看评论列表
- 点赞功能

### 4. 帖子管理
- 查看自己发布的帖子
- 删除自己的帖子

### 5. 帖子评价/评论
- 发表评论
- 回复评论
- 删除评论

### 帖子发布 Service
```java
/**
 * 帖子发布服务实现
 */
@Override
public void publish(PostPublishDTO postPublishDTO, Integer userId) {
    // 1. 敏感词过滤 - 检查标题
    if (postPublishDTO.getTitle() != null &&
        sensitiveWordService.containsSensitiveWord(postPublishDTO.getTitle())) {
        throw new IllegalArgumentException("标题包含敏感词");
    }

    // 2. 敏感词过滤 - 检查内容
    if (postPublishDTO.getContent() != null &&
        sensitiveWordService.containsSensitiveWord(postPublishDTO.getContent())) {
        throw new IllegalArgumentException("内容包含敏感词");
    }

    // 3. 构建帖子对象
    Post post = Post.builder()
            .title(postPublishDTO.getTitle())
            .content(postPublishDTO.getContent())
            .typeId(postPublishDTO.getTypeId())
            .userId(userId)
            .viewCount(0L)
            .likeCount(0L)
            .commentCount(0L)
            .status(1)  // 已发布
            .isTop(0)   // 默认不置顶
            .createTime(LocalDateTime.now())
            .build();

    // 4. 保存帖子
    postMapper.insert(post);

    // 5. 保存帖子图片
    if (postPublishDTO.getImages() != null && !postPublishDTO.getImages().isEmpty()) {
        for (String imageUrl : postPublishDTO.getImages()) {
            PostImage image = PostImage.builder()
                    .postId(post.getId())
                    .imageUrl(imageUrl)
                    .sortOrder(0)
                    .build();
            postImageMapper.insert(image);
        }
    }
}
```

### 推荐算法 Service
```java
/**
 * 推荐服务实现
 * 基于用户行为和帖子热度进行推荐
 */
@Service
public class RecommendationServiceImpl implements RecommendationService {

    @Autowired
    private UserBehaviorMapper userBehaviorMapper;

    @Autowired
    private PostMapper postMapper;

    /**
     * 获取个性化推荐帖子列表
     * @param userId 用户ID
     * @param limit 返回数量
     * @return 推荐帖子列表
     */
    @Override
    public List<Post> getRecommendedPosts(Integer userId, Integer limit) {
        // 1. 获取用户最近浏览的帖子类型偏好
        List<UserBehavior> behaviors = userBehaviorMapper.getRecentBehaviors(userId, 30);

        // 2. 统计用户关注的类型
        Map<Integer, Long> typePreference = behaviors.stream()
                .filter(b -> "view".equals(b.getBehaviorType()))
                .collect(Collectors.groupingBy(UserBehavior::getTargetType, Collectors.counting()));

        // 3. 查询热门帖子
        List<Post> hotPosts = postMapper.getHotPosts(limit * 2);

        // 4. 查询最新帖子
        List<Post> latestPosts = postMapper.getLatestPosts(limit);

        // 5. 根据用户偏好加权排序
        List<Post> result = new ArrayList<>();
        Set<Long> addedIds = new HashSet<>();

        // 优先添加符合用户偏好的热门帖子
        for (Post post : hotPosts) {
            if (result.size() >= limit) break;
            Long prefCount = typePreference.getOrDefault(post.getTypeId(), 0L);
            if (prefCount > 0 || behaviors.isEmpty()) {
                result.add(post);
                addedIds.add(post.getId());
            }
        }

        // 补充最新帖子
        for (Post post : latestPosts) {
            if (result.size() >= limit) break;
            if (!addedIds.contains(post.getId())) {
                result.add(post);
            }
        }

        return result;
    }

    /**
     * 记录用户行为
     * 用于推荐算法学习
     */
    @Override
    public void recordBehavior(Integer userId, String behaviorType, Long targetId, Integer targetType) {
        UserBehavior behavior = UserBehavior.builder()
                .userId(userId)
                .behaviorType(behaviorType)
                .targetId(targetId)
                .targetType(targetType)
                .createTime(LocalDateTime.now())
                .build();
        userBehaviorMapper.insert(behavior);
    }
}
```

## 数据库表

### post_info（帖子表）
```sql
CREATE TABLE `post_info` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '帖子ID',
  `title` varchar(100) NOT NULL COMMENT '帖子标题',
  `content` text NOT NULL COMMENT '帖子内容',
  `type_id` int NOT NULL COMMENT '帖子类型ID',
  `user_id` int NOT NULL COMMENT '发布者用户ID',
  `view_count` bigint DEFAULT 0 COMMENT '浏览量',
  `like_count` bigint DEFAULT 0 COMMENT '点赞数',
  `comment_count` bigint DEFAULT 0 COMMENT '评论数',
  `status` tinyint NOT NULL DEFAULT 1 COMMENT '状态: 0-草稿 1-已发布 2-已删除',
  `is_top` tinyint DEFAULT 0 COMMENT '是否置顶: 0-否 1-是',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  PRIMARY KEY (`id`),
  INDEX `idx_type_id`(`type_id`),
  INDEX `idx_user_id`(`user_id`),
  INDEX `idx_create_time`(`create_time`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='帖子信息表';
```
