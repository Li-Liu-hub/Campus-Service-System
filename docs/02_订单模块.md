# 订单模块

## 模块简介

订单模块是校园服务平台的核心功能，允许学生发布服务需求（如替课、打印、购物等），其他学生可以接单完成。该模块支持普通订单和秒杀订单两种模式。

## 功能清单

### 1. 发布订单
- 选择订单类型（替课、打印、购物、取外卖、快递等）
- 填写服务地址（支持选择常用地址）
- 填写需求描述
- 设置订单酬金（1-10000元）
- 设置要求完成时间
- **可选：开启秒杀订单**（置顶显示，额外¥10费用）

### 2. 订单查询
- 分页查询待接订单
- 按订单号搜索
- 按订单类型筛选
- **秒杀订单自动置顶**

### 3. 接单功能
- 浏览待接订单列表
- 查看订单详情
- 确认接单（显示实际酬金=订单金额+秒杀费）

### 4. 订单管理
- 查看已发布订单
- 查看已接取订单
- 交易历史记录

### 5. 订单完成
- 双方确认完成
- 记录完成时间
- 发送完成通知

### 6. 订单取消
- 待接单状态：发布者可直接取消
- 已接单状态：需双方确认取消

## 订单状态流转

```
待接单(0) ──(被接单)──> 已接单(1) ──(双方确认)──> 已完成(2)
    │                              │
    └──(发布者取消)──> 已取消(3) <──(任意一方取消)──┘
```

| 状态码 | 说明 | 可操作 |
|--------|------|--------|
| 0 | 待接单 | 接单、取消（仅发布者） |
| 1 | 已接单/进行中 | 完成、取消（需双方确认） |
| 2 | 已完成 | 评价 |
| 3 | 已取消 | 无 |

## 秒杀订单特性

- 发布订单时可选择开启秒杀
- 秒杀订单额外收取 ¥10.00 费用
- 秒杀订单在列表中**置顶显示**
- 秒杀订单使用**红色主题UI样式**
- 接单时显示实际酬金（含秒杀费）

### 发布订单 Service
```java
/**
 * 发布订单服务实现
 * 处理订单发布的业务逻辑
 */
@Override
public void publish(OrderPublishDTO orderPublishDTO, Integer userId) {
    // 1. 敏感词过滤 - 检查需求描述
    if (orderPublishDTO.getRequirement() != null &&
        sensitiveWordService.containsSensitiveWord(orderPublishDTO.getRequirement())) {
        List<String> words = sensitiveWordService.getSensitiveWords(orderPublishDTO.getRequirement());
        throw new IllegalArgumentException("订单需求包含敏感词：" + String.join(", ", words));
    }

    // 2. 敏感词过滤 - 检查服务地址
    if (orderPublishDTO.getServiceAddress() != null &&
        sensitiveWordService.containsSensitiveWord(orderPublishDTO.getServiceAddress())) {
        List<String> words = sensitiveWordService.getSensitiveWords(orderPublishDTO.getServiceAddress());
        throw new IllegalArgumentException("服务地址包含敏感词：" + String.join(", ", words));
    }

    // 3. 处理秒杀订单 - 秒杀订单额外加10元
    Integer isSeckill = orderPublishDTO.getIsSeckill();
    if (isSeckill == null) {
        isSeckill = 0;  // 默认普通订单
    }

    BigDecimal seckillFee = BigDecimal.ZERO;
    if (isSeckill == 1) {
        seckillFee = new BigDecimal("10.00");  // 秒杀附加费用
    }

    // 4. 构建订单对象
    Order order = Order.builder()
            .orderNo(generateOrderNo())  // 生成唯一订单号
            .typeId(orderPublishDTO.getTypeId())
            .serviceAddress(orderPublishDTO.getServiceAddress())
            .requirement(orderPublishDTO.getRequirement())
            .contactPhone(orderPublishDTO.getContactPhone())
            .orderAmount(orderPublishDTO.getOrderAmount())
            .orderStatus(StatusConstant.ORDER_STATUS_PENDING)  // 0-待接单
            .userId(userId)
            .createTime(LocalDateTime.now())
            .campusId(orderPublishDTO.getCampusId())
            .requireTime(orderPublishDTO.getRequireTime())
            .isSeckill(isSeckill)
            .seckillFee(seckillFee)
            .build();

    // 5. 保存到数据库
    orderCenterMapper.insert(order);
}

/**
 * 生成唯一订单号
 * 格式: ORD + 时间戳 + 8位随机字符串
 */
private String generateOrderNo() {
    return "ORD" + System.currentTimeMillis() + UUID.randomUUID().toString().substring(0, 8).toUpperCase();
}
```

### 接单 Service
```java
/**
 * 接单服务实现
 * 处理用户接取订单的业务逻辑
 */
@Override
public void accept(Long orderId, Integer userId) {
    // 1. 查询订单
    Order order = orderCenterMapper.getById(orderId);
    if (order == null) {
        throw new OrderBusinessException(MessageConstant.ORDER_NOT_FOUND);
    }

    // 2. 校验：不能接自己的订单
    if (order.getUserId().equals(userId)) {
        throw new OrderBusinessException("不能接取自己的订单");
    }

    // 3. 校验：订单状态必须是待接单
    if (!StatusConstant.ORDER_STATUS_PENDING.equals(order.getOrderStatus())) {
        throw new OrderBusinessException(MessageConstant.ORDER_STATUS_ERROR);
    }

    // 4. 更新订单状态
    order.setAcceptorId(userId);
    order.setOrderStatus(StatusConstant.ORDER_STATUS_ACCEPTED);  // 1-已接单
    orderCenterMapper.update(order);

    // 5. 发送通知给订单发布者
    Notification notification = Notification.builder()
            .userId(order.getUserId())
            .type(3)  // 订单通知
            .title("订单已接单")
            .content("您的订单 " + order.getOrderNo() + " 已被接单，请及时查看")
            .relatedId(orderId)
            .isRead(0)
            .createTime(LocalDateTime.now())
            .build();
    notificationService.createNotification(notification);
}
```

### 取消订单 Service
```java
/**
 * 用户取消订单
 * 根据订单状态不同，处理逻辑不同
 */
@Override
public void cancelByUser(Long orderId, Integer userId) {
    // 1. 查询订单
    Order order = orderCenterMapper.getById(orderId);
    if (order == null) {
        throw new OrderBusinessException(MessageConstant.ORDER_NOT_FOUND);
    }

    // 2. 校验权限：只有发布者或接单者可以取消
    if (!order.getUserId().equals(userId) && !order.getAcceptorId().equals(userId)) {
        throw new OrderBusinessException("您无权取消此订单");
    }

    // 3. 校验：已完成订单不能取消
    if (order.getOrderStatus() == StatusConstant.ORDER_STATUS_COMPLETED) {
        throw new OrderBusinessException("订单已完成，无法取消");
    }

    // 4. 校验：已取消订单不能重复取消
    if (order.getOrderStatus() == StatusConstant.ORDER_STATUS_CANCELLED) {
        throw new OrderBusinessException("订单已取消");
    }

    // 5. 待接单状态：发布者可直接取消
    if (order.getOrderStatus() == StatusConstant.ORDER_STATUS_PENDING) {
        if (order.getUserId().equals(userId)) {
            order.setOrderStatus(StatusConstant.ORDER_STATUS_CANCELLED);
            orderCenterMapper.update(order);
            return;
        } else {
            throw new OrderBusinessException("订单未被接单，无法取消");
        }
    }

    // 6. 已接单状态：需要双方确认才能取消
    // 发布者申请取消
    if (order.getUserId().equals(userId)) {
        if (order.getPublisherCancel() != null && order.getPublisherCancel() == 1) {
            throw new OrderBusinessException("您已申请过取消");
        }
        order.setPublisherCancel(1);
        // 如果接单者也申请了取消，则订单取消成功
        if (order.getAcceptorId() != null && order.getAcceptorCancel() != null && order.getAcceptorCancel() == 1) {
            order.setOrderStatus(StatusConstant.ORDER_STATUS_CANCELLED);
        }
    }
    // 接单者申请取消
    else if (order.getAcceptorId() != null && order.getAcceptorId().equals(userId)) {
        if (order.getAcceptorCancel() != null && order.getAcceptorCancel() == 1) {
            throw new OrderBusinessException("您已申请过取消");
        }
        order.setAcceptorCancel(1);
        // 如果发布者也申请了取消，则订单取消成功
        if (order.getPublisherCancel() != null && order.getPublisherCancel() == 1) {
            order.setOrderStatus(StatusConstant.ORDER_STATUS_CANCELLED);
        }
    }

    orderCenterMapper.update(order);
}
```

### 订单列表组件（秒杀订单置顶）

// 秒杀订单置顶排序
const filteredOrders = computed(() => {
  let result = orders.value;

  // 按类型筛选
  if (filterForm.value.typeId) {
    result = result.filter(order => order.typeId === filterForm.value.typeId);
  }

  // 秒杀订单置顶
  result = [...result].sort((a, b) => {
    if (a.isSeckill === 1 && b.isSeckill !== 1) return -1;
    if (a.isSeckill !== 1 && b.isSeckill === 1) return 1;
    return 0;
  });

  return result;
});
