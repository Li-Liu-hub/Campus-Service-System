# 评论模块

## 模块简介

评论模块为帖子和订单提供评论互动功能，支持发表、回复、删除评论。

## 功能清单

### 1. 发表评论
- 对帖子发表评论
- 评论内容敏感词过滤
- 更新帖子评论数

### 2. 评论回复
- 回复已有评论
- 支持嵌套回复

### 3. 评论管理
- 查看评论列表
- 删除自己的评论
- 帖子作者可删除评论

### 评论发布 Service
```java
/**
 * 评论服务实现
 */
@Service
public class CommentServiceImpl implements CommentService {

    @Autowired
    private CommentMapper commentMapper;

    @Autowired
    private PostMapper postMapper;

    @Autowired
    private SensitiveWordService sensitiveWordService;

    /**
     * 发布评论
     * @param dto 评论发布DTO
     * @param userId 评论用户ID
     */
    @Override
    public void publish(CommentPublishDTO dto, Integer userId) {
        // 1. 敏感词过滤
        if (dto.getContent() != null &&
            sensitiveWordService.containsSensitiveWord(dto.getContent())) {
            throw new IllegalArgumentException("评论内容包含敏感词");
        }

        // 2. 构建评论对象
        Comment comment = Comment.builder()
                .postId(dto.getPostId())
                .parentId(dto.getParentId())  // 父评论ID，为空则为顶层评论
                .userId(userId)
                .repliedUserId(dto.getRepliedUserId())  // 被回复用户ID
                .content(dto.getContent())
                .likeCount(0L)
                .createTime(LocalDateTime.now())
                .build();

        // 3. 保存评论
        commentMapper.insert(comment);

        // 4. 更新帖子评论数
        postMapper.incrementCommentCount(dto.getPostId());
    }

    /**
     * 获取帖子评论列表
     * @param postId 帖子ID
     * @return 评论列表（树形结构）
     */
    @Override
    public List<CommentVO> getByPostId(Long postId) {
        // 1. 查询顶层评论
        List<Comment> topComments = commentMapper.getTopComments(postId);

        // 2. 递归获取回复评论
        List<CommentVO> result = new ArrayList<>();
        for (Comment comment : topComments) {
            CommentVO vo = convertToVO(comment);
            vo.setReplies(getReplies(comment.getId()));
            result.add(vo);
        }

        return result;
    }

    /**
     * 递归获取回复评论
     */
    private List<CommentVO> getReplies(Long parentId) {
        List<Comment> replies = commentMapper.getByParentId(parentId);
        List<CommentVO> result = new ArrayList<>();
        for (Comment reply : replies) {
            CommentVO vo = convertToVO(reply);
            vo.setReplies(getReplies(reply.getId()));  // 递归获取子回复
            result.add(vo);
        }
        return result;
    }

    private CommentVO convertToVO(Comment comment) {
        CommentVO vo = new CommentVO();
        BeanUtils.copyProperties(comment, vo);
        return vo;
    }
}
```

## 数据库表

### comment_info（评论表）
```sql
CREATE TABLE `comment_info` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '评论ID',
  `post_id` bigint NOT NULL COMMENT '帖子ID',
  `parent_id` bigint DEFAULT NULL COMMENT '父评论ID（用于回复）',
  `user_id` int NOT NULL COMMENT '评论用户ID',
  `replied_user_id` int DEFAULT NULL COMMENT '被评论用户ID',
  `content` text NOT NULL COMMENT '评论内容',
  `like_count` bigint DEFAULT 0 COMMENT '点赞数',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (`id`),
  INDEX `idx_post_id`(`post_id`),
  INDEX `idx_parent_id`(`parent_id`),
  INDEX `idx_user_id`(`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='评论信息表';
```
