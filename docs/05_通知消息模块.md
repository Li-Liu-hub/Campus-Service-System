# 通知消息模块

## 模块简介

通知消息模块负责向用户推送各类系统通知和订单状态变更通知，支持WebSocket实时推送。

## 功能清单

### 1. 系统通知
- 管理员发布通知
- 全站广播通知

### 2. 订单通知
- 订单被接单通知
- 订单完成通知
- 订单取消通知

### 3. 通知管理
- 查看通知列表
- 标记已读
- 删除通知

### 4. 实时推送
- WebSocket实时推送
- 未读消息数量提醒

## 通知类型

| 类型ID | 说明 |
|--------|------|
| 1 | 系统公告 |
| 2 | 互动通知 |
| 3 | 订单通知 |

## 技术实现

### 通知实体类
```java
/**
 * 通知实体
 * 对应数据库 notification 表
 */
@Data
@Builder
@NoArgsConstructor
@AllArgsConstructor
public class Notification implements Serializable {
    private static final long serialVersionUID = 1L;

    /** 通知ID */
    private Long id;

    /** 接收用户ID */
    private Integer userId;

    /** 通知类型: 1-系统公告 2-互动通知 3-订单通知 */
    private Integer type;

    /** 通知标题 */
    private String title;

    /** 通知内容 */
    private String content;

    /** 关联ID（帖子ID/订单ID等） */
    private Long relatedId;

    /** 是否已读: 0-未读 1-已读 */
    private Integer isRead;

    /** 创建时间 */
    private LocalDateTime createTime;
}
```

### 通知 Service
```java
/**
 * 通知服务实现
 */
@Service
public class NotificationServiceImpl implements NotificationService {

    @Autowired
    private NotificationMapper notificationMapper;

    @Autowired
    private ChatWebSocketHandler webSocketHandler;

    /**
     * 创建通知
     * @param notification 通知对象
     */
    @Override
    public void createNotification(Notification notification) {
        // 1. 保存到数据库
        notificationMapper.insert(notification);

        // 2. WebSocket 实时推送
        try {
            WebSocketMessageDTO message = WebSocketMessageDTO.builder()
                    .type("notification")
                    .data(notification)
                    .build();
            webSocketHandler.sendMessageToUser(notification.getUserId().longValue(), message);
        } catch (IOException e) {
            log.error("WebSocket推送失败", e);
        }
    }

    /**
     * 广播通知（系统公告）
     * @param notification 通知内容
     */
    @Override
    public void broadcastNotification(Notification notification) {
        // 查询所有用户ID，逐一发送
        // 实际生产环境应使用消息队列批量处理
        // 此处简化处理
        List<Integer> allUserIds = userMapper.getAllUserIds();
        for (Integer userId : allUserIds) {
            Notification singleNotification = Notification.builder()
                    .userId(userId)
                    .type(1)  // 系统公告
                    .title(notification.getTitle())
                    .content(notification.getContent())
                    .isRead(0)
                    .createTime(LocalDateTime.now())
                    .build();
            createNotification(singleNotification);
        }
    }

    /**
     * 获取用户通知列表
     */
    @Override
    public List<Notification> getByUserId(Integer userId, Integer page, Integer pageSize) {
        return notificationMapper.getByUserId(userId, (page - 1) * pageSize, pageSize);
    }

    /**
     * 标记通知已读
     */
    @Override
    public void markAsRead(Long notificationId, Integer userId) {
        notificationMapper.markAsRead(notificationId, userId);
    }

    /**
     * 获取未读通知数量
     */
    @Override
    public Long getUnreadCount(Integer userId) {
        return notificationMapper.getUnreadCount(userId);
    }
}
```

### WebSocket 配置
```java
/**
 * WebSocket 配置类
 * 配置 WebSocket 端点和拦截器
 */
@Configuration
@EnableWebSocket
public class WebSocketConfig implements WebSocketConfigurer {

    @Autowired
    private ChatWebSocketHandler chatWebSocketHandler;

    @Autowired
    private WebSocketHandshakeInterceptor handshakeInterceptor;

    /**
     * 注册 WebSocket 端点
     */
    @Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry registry) {
        registry.addHandler(chatWebSocketHandler, "/ws/chat")
                .addInterceptors(handshakeInterceptor)
                .setAllowedOrigins("*");  // 生产环境应限制来源
    }
}
```

### WebSocket 处理器
```java
/**
 * WebSocket 聊天处理器
 * 处理实时消息推送
 */
@Component
public class ChatWebSocketHandler extends TextWebSocketHandler {

    /** 用户ID -> Session 映射 */
    private static final Map<Long, WebSocketSession> SESSIONS = new ConcurrentHashMap<>();

    /**
     * 客户端连接成功
     */
    @Override
    public void afterConnectionEstablished(WebSocketSession session) {
        Long userId = (Long) session.getAttributes().get("userId");
        if (userId != null) {
            SESSIONS.put(userId, session);
            log.info("用户 {} 连接 WebSocket", userId);
        }
    }

    /**
     * 客户端断开连接
     */
    @Override
    public void afterConnectionClosed(WebSocketSession session, CloseStatus status) {
        Long userId = (Long) session.getAttributes().get("userId");
        if (userId != null) {
            SESSIONS.remove(userId);
            log.info("用户 {} 断开 WebSocket", userId);
        }
    }

    /**
     * 发送消息给指定用户
     */
    public void sendMessageToUser(Long userId, WebSocketMessageDTO message) {
        WebSocketSession session = SESSIONS.get(userId);
        if (session != null && session.isOpen()) {
            try {
                String json = new ObjectMapper().writeValueAsString(message);
                session.sendMessage(new TextMessage(json));
            } catch (IOException e) {
                log.error("发送消息失败", e);
            }
        }
    }

    /**
     * 广播消息给所有在线用户
     */
    public void broadcast(WebSocketMessageDTO message) {
        SESSIONS.values().forEach(session -> {
            if (session.isOpen()) {
                try {
                    String json = new ObjectMapper().writeValueAsString(message);
                    session.sendMessage(new TextMessage(json));
                } catch (IOException e) {
                    log.error("广播消息失败", e);
                }
            }
        });
    }
}
```

## 数据库表

### notification（通知表）
```sql
CREATE TABLE `notification` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '通知ID',
  `user_id` int NOT NULL COMMENT '接收用户ID',
  `type` tinyint NOT NULL COMMENT '通知类型: 1-系统公告 2-互动通知 3-订单通知',
  `title` varchar(100) NOT NULL COMMENT '通知标题',
  `content` text COMMENT '通知内容',
  `related_id` bigint DEFAULT NULL COMMENT '关联ID',
  `is_read` tinyint NOT NULL DEFAULT 0 COMMENT '是否已读: 0-未读 1-已读',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (`id`),
  INDEX `idx_user_id`(`user_id`),
  INDEX `idx_is_read`(`is_read`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='通知消息表';
```
